# python-exercises
Uma coleção de exercícios e pequenos projetos em Python para iniciantes, com foco nos fundamentos da programação.
A collection of beginner Python exercises and small projects focused on programming fundamentals.

## 17/07/2026 – Calculadora Simplificada

Este foi o primeiro projeto de programação que desenvolvi. Procurei reproduzir o funcionamento básico de uma calculadora convencional, seguindo a sequência de entrada de dados e utilizando os operadores +, -, * e /.

Um aspecto que achei interessante foi perceber como conceitos de lógica de proposições, que já conhecia por estudar para concursos, também são aplicados na programação, especialmente na construção de estruturas condicionais e no controle do fluxo de execução.

## 2026/07/17 – Simple Calculator

This was the first programming project I built. I tried to mimic the basic behavior of a regular calculator by following the sequence of user input and implementing the four basic arithmetic operators (`+`, `-`, `*`, and `/`).

One interesting realization was that some concepts from propositional logic, which I had previously studied while preparing for civil service exams, also apply to programming, especially when working with conditional statements and program flow.
