# python-exercises
Uma coleção de exercícios e pequenos projetos em Python para iniciantes, com foco nos fundamentos da programação.
A collection of beginner Python exercises and small projects focused on programming fundamentals.

## 17/07/2026 – Calculadora Simplificada

Este foi o primeiro projeto de programação que desenvolvi. Procurei reproduzir o funcionamento básico de uma calculadora convencional, seguindo a sequência de entrada de dados e utilizando os operadores +, -, * e /.

Um aspecto que achei interessante foi perceber como conceitos de lógica de proposições, que já conhecia por estudar para concursos, também são aplicados na programação, especialmente na construção de estruturas condicionais e no controle do fluxo de execução.

- Aprendi que as variáveis são sucedidas de '=', onde estará especificado o seu valor;
- que '==' compara valores, retornando um dado booleano; 
- que um booleano é um dado que recebe valor de verdadeiro ou falso;
- if é a condicional se, semelhante à lógica de proposição e é usada para analisar algum parâmetro foi satisfeito (no caso, qual foi a operação escolhida) e como preceder caso tenha sido satisfeito;
- elif como subsidiária do if ou de elif anterior, ou seja, agindo quando os anteriores não tenham sido satisfeitos; 
- que print exibe alguma informação no terminal; 
- que aspas duplas são usadas para string, contendo qualquer sequência de caracteres; 
- f antes de string possibilita a inserção de uma variável na string; 
- que para inserir uma variável na string é necessário usar chaves {}.

## 2026/07/17 – Simple Calculator

This was the first programming project I built. I tried to mimic the basic behavior of a regular calculator by following the sequence of user input and implementing the four basic arithmetic operators (`+`, `-`, `*`, and `/`).

One interesting realization was that some concepts from propositional logic, which I had previously studied while preparing for civil service exams, also apply to programming, especially when working with conditional statements and program flow.

- I learned that variables are followed by `=`, where their value is specified;
- that `==` compares values, returning a boolean value;
- that a boolean is a data type that can have the value `True` or `False`;
- that `if` is the "if" conditional, similar to propositional logic, and is used to check whether a given condition has been satisfied (in this project, which operation was chosen) and determine what happens if it is;
- that `elif` acts as a continuation of the previous `if` or `elif`, being evaluated only when the previous conditions are not satisfied;
- that `print` displays information in the terminal;
- that double quotes are used for strings, which can contain any sequence of characters;
- that placing `f` before a string allows variables to be inserted into it;
- that variables are inserted into an f-string using curly braces `{}`.
